Title: Hospital Static Web App – CarePlus Hospital

Description:
This is a responsive static web application for a hospital website built using HTML, CSS, and JavaScript. It showcases a modern, clean layout suitable for healthcare services. The project is designed as a portfolio-level frontend project, demonstrating layout design, styling, and basic interactivity.

Features:

Hero Section: Attractive call-to-action with “Book Appointment” button.

Departments Section: Highlights multiple hospital departments with card UI.

Sticky Header: Navigation bar remains visible on scroll.

Interactive Buttons: Hover effects and alert pop-ups using JavaScript.

Responsive Design: Works across desktops, tablets, and mobile screens.

Tech Stack:

HTML5

CSS3 (flexbox layout, responsive design)

Vanilla JavaScript (DOM manipulation, event handling)

Use Case:
Ideal as a portfolio project for demonstrating frontend development skills and understanding of static web applications.

Future Enhancements:

Connect “Book Appointment” button to a backend service.

Add additional sections like Doctors, Services, and Contact Form.

Integrate animations and accessibility improvements.
